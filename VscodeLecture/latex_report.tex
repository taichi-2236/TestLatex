\section{はじめに}
このレポートはVSCode講座の練習用レポートです．ここでは\LaTeX を例にして扱いますが，他の言語のプログラムでも十分に応用が利きます．この講座を通してVSCodeの機能を利用し，実践を踏まえて使いこなせるようになることを期待します．

\section{VSCodeについて}
VSCodeはMicrosoftが開発・提供するコードエディタです．VSCodeでコーディングを行うことには以下のような利点があります．
\subsubsection{利点}
\begin{enumerate}
  \item コードが見やすい
  \item コードを読む上での補助機能がある
  \item 言語ごとに細かな設定ができる
  \item スニペットが使える
  \item 多様なショートカットが使える
  \item 拡張機能が豊富
  \item 他プログラム開発ソフトとの高い互換性
  \item 細かいカスタムができる
\end{enumerate}
